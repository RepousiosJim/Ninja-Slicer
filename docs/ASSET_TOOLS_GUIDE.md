# ğŸ’¡ PRO TIPS

## For AI Sprite Generators

1. **Be Specific:** Include "pixel art", exact dimensions, "transparent background", "sprite sheet"

2. **Style Keywords:** "16-bit", "dark fantasy", "gothic horror", "Castlevania style"

3. **Animation Tips:** Generate static poses first, then request variations for animation frames

4. **Consistency:** Use the same tool for all monsters to maintain style consistency

## For Sound Generation

1. **Variations:** Always create 3-5 variations of common sounds (slashes)

2. **Layer Sounds:** Combine jsfxr output with ElevenLabs for richer sounds

3. **Format:** Export as .wav, convert to .ogg for web (smaller files)

## Color Palette Reference
```
Primary:     #8b0000, #aa0000 (Dark Reds)
Secondary:   #4a0080, #6a0dad (Dark Purples)  
Accent:      #ffd700 (Gold)
Background:  #1a1a2e, #2a2a4e (Dark Blue-Gray)
```
