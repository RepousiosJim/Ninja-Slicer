# UI Assets

Place your UI sprite images in this folder.

## Required Images

### UI Elements
- `button_normal.png` - Normal button state (recommended size varies)
- `button_hover.png` - Hover button state
- `button_pressed.png` - Pressed button state
- `button_disabled.png` - Disabled button state
- `panel_background.png` - Panel background (recommended size varies)
- `panel_border.png` - Panel border (recommended size varies)
- `heart_full.png` - Full heart icon (32x32 recommended)
- `heart_empty.png` - Empty heart icon (32x32 recommended)
- `star_empty.png` - Empty star icon (32x32 recommended)
- `star_filled.png` - Filled star icon (32x32 recommended)
- `pause_icon.png` - Pause button icon (32x32 recommended)
- `settings_icon.png` - Settings icon (32x32 recommended)
- `shop_icon.png` - Shop icon (32x32 recommended)
- `character_icon.png` - Character icon (32x32 recommended)
- `inventory_icon.png` - Inventory icon (32x32 recommended)
- `leaderboard_icon.png` - Leaderboard icon (32x32 recommended)

## Notes
- All sprites should be PNG format with transparency
- Use pixel art style for consistency
- Buttons should have 4 states (normal, hover, pressed, disabled)
- Icons should be 32x32 pixels for consistency
