# Enemies Assets

Place your enemy sprite images in this folder.

## Required Images

### Monsters
- `zombie.png` - Zombie sprite (64x64 recommended)
- `vampire.png` - Vampire sprite (64x64 recommended)
- `ghost.png` - Ghost sprite (64x64 recommended)
- `villager.png` - Villager sprite (48x48 recommended)

### Bosses
- `boss_gravetitan.png` - Grave Titan boss sprite (128x128 recommended)
- `boss_horseman.png` - Headless Horseman boss sprite (128x128 recommended)
- `boss_vampirelord.png` - Vampire Lord boss sprite (128x128 recommended)
- `boss_phantomking.png` - Phantom King boss sprite (128x128 recommended)
- `boss_demonoverlord.png` - Demon Overlord boss sprite (128x128 recommended)

## Notes
- All sprites should be PNG format with transparency
- Recommended size: 64x64 for monsters, 128x128 for bosses
- Use pixel art style for consistency
- Include idle, attack, and death animations if possible
