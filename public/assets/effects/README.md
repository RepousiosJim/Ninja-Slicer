# Effects Assets

Place your effect sprite images in this folder.

## Required Images

### Slash Effects
- `slash_trail.png` - Slash trail effect (64x64 recommended)
- `slash_hit.png` - Slash hit effect (64x64 recommended)
- `slash_critical.png` - Critical hit effect (64x64 recommended)

### Monster Effects
- `monster_death.png` - Monster death effect (64x64 recommended)
- `monster_burn.png` - Burn effect (64x64 recommended)
- `monster_stun.png` - Stun effect (64x64 recommended)
- `monster_slow.png` - Slow effect (64x64 recommended)

### Boss Effects
- `boss_phasechange.png` - Boss phase change effect (128x128 recommended)
- `boss_death.png` - Boss death effect (128x128 recommended)
- `boss_attack.png` - Boss attack effect (128x128 recommended)

### Power-Up Effects
- `powerup_collected.png` - Power-up collected effect (64x64 recommended)
- `powerup_activated.png` - Power-up activated effect (64x64 recommended)

### UI Effects
- `star_earned.png` - Star earned effect (64x64 recommended)
- `combo_milestone.png` - Combo milestone effect (64x64 recommended)
- `score_popup.png` - Score popup effect (64x64 recommended)

## Notes
- All sprites should be PNG format with transparency
- Use pixel art style for consistency
- Include animation frames if possible
- Effects should be subtle and not overwhelming
