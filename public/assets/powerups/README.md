# Power-Ups Assets

Place your power-up sprite images in this folder.

## Required Images

### Power-Ups (4 total)
- `powerup_slowmotion.png` - Slow Motion power-up sprite (64x64 recommended)
- `powerup_frenzy.png` - Frenzy power-up sprite (64x64 recommended)
- `powerup_shield.png` - Shield power-up sprite (64x64 recommended)
- `powerup_soulmagnet.png` - Soul Magnet power-up sprite (64x64 recommended)

## Notes
- All sprites should be PNG format with transparency
- Recommended size: 64x64 pixels
- Use pixel art style for consistency
- Include idle and collected animations if possible
- Each power-up should have a distinct color scheme:
  - Slow Motion: Cyan/Blue
  - Frenzy: Orange/Red
  - Shield: Green
  - Soul Magnet: Purple
