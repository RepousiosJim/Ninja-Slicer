# Weapons Assets

Place your weapon sprite images in this folder.

## Required Images

### Weapons (6 total)
- `weapon_basicsword.png` - Basic Sword sprite (64x64 recommended)
- `weapon_silverblade.png` - Silver Blade sprite (64x64 recommended)
- `weapon_holycrossblade.png` - Holy Cross Blade sprite (64x64 recommended)
- `weapon_firesword.png` - Fire Sword sprite (64x64 recommended)
- `weapon_iceblade.png` - Ice Blade sprite (64x64 recommended)
- `weapon_lightningkatana.png` - Lightning Katana sprite (64x64 recommended)

## Notes
- All sprites should be PNG format with transparency
- Recommended size: 64x64 pixels
- Use pixel art style for consistency
- Include idle and attack animations if possible
- Each weapon should have 3 tiers (can use same sprite with color variations)
