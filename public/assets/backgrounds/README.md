# Backgrounds Assets

Place your background images in this folder.

## Required Images

### World Backgrounds (5 total)
- `world1_background.png` - World 1 background (1280x720 recommended)
- `world2_background.png` - World 2 background (1280x720 recommended)
- `world3_background.png` - World 3 background (1280x720 recommended)
- `world4_background.png` - World 4 background (1280x720 recommended)
- `world5_background.png` - World 5 background (1280x720 recommended)

### Other Backgrounds
- `mainmenu_background.png` - Main menu background (1280x720 recommended)
- `shop_background.png` - Shop background (1280x720 recommended)
- `character_background.png` - Character scene background (1280x720 recommended)
- `leaderboard_background.png` - Leaderboard background (1280x720 recommended)

## Notes
- All backgrounds should be PNG format
- Recommended size: 1280x720 pixels (game resolution)
- Use pixel art style for consistency
- Each world should have a distinct theme:
  - World 1: Graveyard (dark green/brown tones)
  - World 2: Haunted Forest (dark blue/purple tones)
  - World 3: Vampire Castle (dark red/black tones)
  - World 4: Ghost Realm (dark purple/gray tones)
  - World 5: Demon Hell (dark red/orange tones)
